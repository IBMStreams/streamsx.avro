# TupleToAvroSample

## Description

This sample demonstrates how you can use the TupleToAvro operator from the com.ibm.streamsx.avro toolkit.


## Use

Build application:

`make`

Run standalone:

`./output/bin/standalone`

You can examine the output in the console.


Clean:

`make clean`


## Utilized Toolkits

 - com.ibm.streamsx.avro
